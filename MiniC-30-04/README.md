#  # Proyecto Compiladores

Este proyecto es un compilador para el lenguaje MiniC, desarrollado en C utilizando las herramientas Bison y Flex.

## Requisitos

- GCC
- Bison
- Flex

## Instalación

Para compilar el proyecto, sigue estos pasos:

1. Clona el repositorio en tu máquina local.
2. Navega hasta el directorio del proyecto.
3. Ejecuta el comando `make` para compilar el proyecto.

## Uso

Para ejecutar el compilador con un archivo de entrada, utiliza el siguiente comando:

```bash
./miniC nombre_del_archivo
